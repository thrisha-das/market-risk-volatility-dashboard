# 📊 Market Risk & Volatility Dashboard

## 📌 Project Overview

This project analyzes financial market data to measure risk and volatility using quantitative techniques.

## 🔍 Key Features

* Stock Price Trend Analysis
* Volatility Measurement
* Value at Risk (VaR) Calculation
* Monte Carlo Simulation

## 🛠 Tools Used

* Python (Pandas, NumPy, Matplotlib)
* Power BI
* Data Analytics

## 📁 Files Included

* finance_data.csv
* analysis.ipynb
* dashboard.png
* market_risk_dashboard.pbix

## 📈 Output

Interactive Power BI dashboard showing financial risk insights.
